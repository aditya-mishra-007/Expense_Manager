# 💰 Expense Manager

A full-stack MERN (MongoDB, Express, React, Node.js) application for tracking daily expenses with a modern, glassmorphism UI.

🚀 **Live Demo:** [https://expense-manager-frontend-3b6l.onrender.com](https://expense-manager-frontend-3b6l.onrender.com)

---

## ✨ Features
* **User Authentication:** Secure Login and Registration using JWT and Bcrypt.
* **Expense Tracking:** Add, view, and delete expenses in real-time.
* **Dashboard:** Visual summaries of total spending and entry counts.
* **Responsive Design:** Fully optimized for mobile and desktop views.
* **Modern UI:** Bento-style layout with dark mode aesthetics.

## 🛠️ Tech Stack
* **Frontend:** React, Vite, Tailwind CSS / CSS Modules
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Deployment:** Render
